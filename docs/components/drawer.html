Drawer

A panel that slides in from the edge of the screen with a drag handle. Ideal for mobile-friendly navigation or forms.

Visual Variance Note

The plank-drawer web component allows up to 2% visual variance compared to the React drawer. This is due to differences between the native vaul library (used in React) and our Lit implementation. The functionality and layout are identical, but minor subpixel rendering differences may occur.

Basic Usage

By default, the drawer slides in from the bottom with a drag handle.

With Close Button

Unlike Sheet, Drawer does not show a close button by default. Use show-close-button to add one.

Drawer vs Sheet

Both Drawer and Sheet are panel components that slide from screen edges. Here's when to use each:

Feature Drawer Sheet
Drag handle Yes (bottom only) No
Close button Optional (off by default) Yes (on by default)
Best for Mobile interactions, bottom sheets Desktop side panels, settings
Text alignment Centered (top/bottom) Left-aligned

API

plank-drawer

Property Type Default Description
open boolean false The controlled open state

Events

Event Detail Description
open-change { open: boolean } Fired when the open state changes

plank-drawer-trigger

The element that triggers the drawer. Opens drawer on click.

plank-drawer-content

Property Type Default Description
direction "top" | "right" | "bottom" | "left" "bottom" Which edge to slide from
show-close-button boolean false Whether to show the close button
class string "" Additional CSS classes

plank-drawer-close

A button that closes the drawer when clicked.

plank-drawer-header

Container for title and description. Text is centered for top/bottom drawers.

plank-drawer-footer

Container for action buttons. Uses mt-auto to stick to the bottom.

plank-drawer-title

The drawer title. Used for aria-labelledby.

plank-drawer-description

The drawer description. Used for aria-describedby.

Accessibility

  • Drawer content has role="dialog"
  • Trigger has aria-haspopup="dialog"
  • Trigger has aria-expanded to indicate open state
  • Trigger has aria-controls pointing to the drawer when open
  • Drawer has aria-labelledby pointing to the title
  • Drawer has aria-describedby pointing to the description
  • Closes on Escape key
  • Closes when clicking on overlay
  • Modal blocks interaction with content behind
