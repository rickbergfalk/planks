Sheet

A panel that slides in from the edge of the screen. Useful for navigation, forms, or supplementary content.

Basic Usage

By default, the sheet slides in from the right side.

Open Sheet Edit profile Make changes to your profile here. Click save when you're done.

Sheet content goes here.

Save changes 
<hal-sheet>
  <hal-sheet-trigger>
    <hal-button variant="outline">Open Sheet</hal-button>
  </hal-sheet-trigger>
  <hal-sheet-content>
    <hal-sheet-header>
      <hal-sheet-title>Edit profile</hal-sheet-title>
      <hal-sheet-description>
        Make changes to your profile here. Click save when you're done.
      </hal-sheet-description>
    </hal-sheet-header>
    <div class="py-4">
      <p class="text-sm">Sheet content goes here.</p>
    </div>
    <hal-sheet-footer>
      <hal-button>Save changes</hal-button>
    </hal-sheet-footer>
  </hal-sheet-content>
</hal-sheet>

Side Variations

Use the side attribute to control which edge the sheet slides from.

Right (Default) Right Sheet This sheet slides in from the right. Left Left Sheet This sheet slides in from the left. Top Top Sheet This sheet slides in from the top. Bottom Bottom Sheet This sheet slides in from the bottom. 
<hal-sheet-content side="left">
  ...
</hal-sheet-content>

<hal-sheet-content side="top">
  ...
</hal-sheet-content>

<hal-sheet-content side="bottom">
  ...
</hal-sheet-content>

Without Close Button

Use show-close-button="false" to hide the default close button.

Open Sheet No Close Button This sheet has no default close button. Press Escape or click outside to close. 
<hal-sheet-content show-close-button="false">
  ...
</hal-sheet-content>

API

hal-sheet

Property Type Default Description
open boolean false The controlled open state

Events

Event Detail Description
open-change { open: boolean } Fired when the open state changes

hal-sheet-trigger

The element that triggers the sheet. Opens sheet on click.

hal-sheet-content

Property Type Default Description
side "top" | "right" | "bottom" | "left" "right" Which edge to slide from
show-close-button boolean true Whether to show the close button
class string "" Additional CSS classes

hal-sheet-close

A button that closes the sheet when clicked.

hal-sheet-header

Container for title and description. Provides padding and spacing.

hal-sheet-footer

Container for action buttons. Uses mt-auto to stick to the bottom.

hal-sheet-title

The sheet title. Used for aria-labelledby.

hal-sheet-description

The sheet description. Used for aria-describedby.

Accessibility

  • Sheet content has role="dialog"
  • Trigger has aria-haspopup="dialog"
  • Trigger has aria-expanded to indicate open state
  • Trigger has aria-controls pointing to the sheet when open
  • Sheet has aria-labelledby pointing to the title
  • Sheet has aria-describedby pointing to the description
  • Closes on Escape key
  • Closes when clicking on overlay
  • Modal blocks interaction with content behind
