Tabs

A set of layered sections of content that display one panel at a time.

React vs Web Component

Basic Usage

Account Password
Make changes to your account here. Click save when you're done.
Change your password here. After saving, you'll be logged out. 
<hal-tabs value="account">
  <hal-tabs-list>
    <hal-tabs-trigger value="account">Account</hal-tabs-trigger>
    <hal-tabs-trigger value="password">Password</hal-tabs-trigger>
  </hal-tabs-list>
  <hal-tabs-content value="account">
    Account content...
  </hal-tabs-content>
  <hal-tabs-content value="password">
    Password content...
  </hal-tabs-content>
</hal-tabs>

API

hal-tabs

Property Type Default Description
value string "" The value of the currently selected tab

Events

Event Detail Description
value-change { value: string } Fired when the selected tab changes

hal-tabs-list

Container for tab triggers. Has role="tablist" for accessibility.

hal-tabs-trigger

Property Type Default Description
value string "" Unique identifier matching a content panel
disabled boolean false Prevents selection of this tab

hal-tabs-content

Property Type Default Description
value string "" Unique identifier matching a trigger

Accessibility

  • Tab list has role="tablist"
  • Each trigger has role="tab" with aria-selected state
  • Each panel has role="tabpanel" with aria-labelledby pointing to its trigger
  • Triggers have aria-controls pointing to their panels
  • Keyboard navigation with Arrow Left/Right to move between tabs
  • Space and Enter keys activate the focused tab
  • Focus wraps from last to first tab and vice versa
