Collapsible

An interactive component which expands/collapses a panel.

Basic Usage

@peduarte starred 3 repositories Toggle
@radix-ui/primitives
@radix-ui/colors
@stitches/react 
<plank-collapsible class="flex flex-col gap-2">
  <div class="flex items-center justify-between gap-4">
    <span class="text-sm font-semibold">@peduarte starred 3 repositories</span>
    <plank-collapsible-trigger>
      <plank-button variant="ghost" size="sm">Toggle</plank-button>
    </plank-collapsible-trigger>
  </div>
  <div class="rounded-md border px-4 py-2 font-mono text-sm">
    @radix-ui/primitives
  </div>
  <plank-collapsible-content class="flex flex-col gap-2">
    <div class="rounded-md border px-4 py-2 font-mono text-sm">
      @radix-ui/colors
    </div>
    <div class="rounded-md border px-4 py-2 font-mono text-sm">
      @stitches/react
    </div>
  </plank-collapsible-content>
</plank-collapsible>

API

plank-collapsible

Property Type Default Description
open boolean false The controlled open state
disabled boolean false Prevents toggling

Events

Event Detail Description
open-change { open: boolean } Fired when the open state changes

plank-collapsible-trigger

The button that toggles the collapsible. Automatically sets up ARIA attributes (role="button", aria-expanded, aria-controls).

plank-collapsible-content

The collapsible content. Hidden when closed, visible when open. Automatically gets an id for aria-controls.

Accessibility

  • Trigger has role="button" with aria-expanded state
  • Trigger has aria-controls pointing to the content's auto-generated id
  • Supports keyboard activation (Space and Enter keys)
  • Content uses hidden attribute when collapsed
