Avatar

An image element with a fallback for representing the user.

Basic Usage

CN 
<plank-avatar>
  <plank-avatar-image src="..." alt="@username"></plank-avatar-image>
  <plank-avatar-fallback>CN</plank-avatar-fallback>
</plank-avatar>

Fallback Only

CN Initials
AB Initials 
<plank-avatar>
  <plank-avatar-fallback>CN</plank-avatar-fallback>
</plank-avatar>

Custom Shapes

CN Circle (default)
ER Rounded
SQ Square 
<plank-avatar>...</plank-avatar>
<plank-avatar class="rounded-lg">...</plank-avatar>
<plank-avatar class="rounded-none">...</plank-avatar>

Stacked Avatars

A B C 
<div class="flex -space-x-2">
  <plank-avatar class="ring-2 ring-background">...</plank-avatar>
  <plank-avatar class="ring-2 ring-background">...</plank-avatar>
  <plank-avatar class="ring-2 ring-background">...</plank-avatar>
</div>

API

plank-avatar

The container component. Contains the image and fallback.

plank-avatar-image

Property Type Description
src string The image source URL
alt string Alternative text for the image

plank-avatar-fallback

Displays fallback content (typically initials) when the image is not available. Children are rendered as the fallback content.
