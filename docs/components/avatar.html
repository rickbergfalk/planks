Avatar

An image element with a fallback for representing the user.

React vs Web Component

Basic Usage

CN 
<hal-avatar>
  <hal-avatar-image src="..." alt="@username"></hal-avatar-image>
  <hal-avatar-fallback>CN</hal-avatar-fallback>
</hal-avatar>

Fallback Only

CN Initials
AB Initials

API

hal-avatar

The container component. Contains the image and fallback.

hal-avatar-image

Property Type Description
src string The image source URL
alt string Alternative text for the image

hal-avatar-fallback

Displays fallback content (typically initials) when the image is not available. Children are rendered as the fallback content.
