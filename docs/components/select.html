Select

A dropdown menu for selecting a single value from a list of options.

Visual Variance Note

The hal-select web component allows up to 2.5% visual variance compared to the React select. This is due to Radix's "item-aligned" positioning mode which overlays the selected item over the trigger. Our web component uses standard floating positioning which displays the dropdown below the trigger. The functionality and styling are identical.

Basic Usage

A simple select with a list of options.

Apple Banana Orange Grape Mango 
<hal-select>
  <hal-select-trigger class="w-[180px]">
    <hal-select-value placeholder="Select a fruit"></hal-select-value>
  </hal-select-trigger>
  <hal-select-content>
    <hal-select-item value="apple">Apple</hal-select-item>
    <hal-select-item value="banana">Banana</hal-select-item>
    <hal-select-item value="orange">Orange</hal-select-item>
  </hal-select-content>
</hal-select>

With Default Value

Use the value attribute to set a default selected value.

Apple Banana Orange 
<hal-select value="banana">
  ...
</hal-select>

With Groups

Organize options into groups with labels.

Fruits Apple Banana Orange Vegetables Carrot Broccoli Spinach 
<hal-select-content>
  <hal-select-group>
    <hal-select-label>Fruits</hal-select-label>
    <hal-select-item value="apple">Apple</hal-select-item>
    <hal-select-item value="banana">Banana</hal-select-item>
  </hal-select-group>
  <hal-select-separator></hal-select-separator>
  <hal-select-group>
    <hal-select-label>Vegetables</hal-select-label>
    <hal-select-item value="carrot">Carrot</hal-select-item>
  </hal-select-group>
</hal-select-content>

Small Size

Use size="sm" on the trigger for a smaller select.

Apple Banana Orange 
<hal-select-trigger size="sm" class="w-[180px]">
  ...
</hal-select-trigger>

Disabled

Use the disabled attribute to prevent interaction.

Apple Banana 
<hal-select disabled>
  ...
</hal-select>

Disabled Items

Individual items can be disabled.

Apple Banana (Out of stock) Orange 
<hal-select-item value="banana" disabled>Banana (Out of stock)</hal-select-item>

API

hal-select

Property Type Default Description
value string "" The selected value
open boolean false The controlled open state
disabled boolean false Whether the select is disabled

Events

Event Detail Description
value-change { value: string } Fired when the selected value changes
open-change { open: boolean } Fired when the open state changes

hal-select-trigger

Property Type Default Description
size "sm" | "default" "default" The trigger size
class string "" Additional CSS classes

hal-select-value

Property Type Default Description
placeholder string "" Placeholder text when no value is selected

hal-select-content

Property Type Default Description
side "top" | "right" | "bottom" | "left" "bottom" Preferred side to position the content
align "start" | "center" | "end" "start" Alignment relative to trigger
sideOffset number 4 Distance from trigger in pixels
class string "" Additional CSS classes

hal-select-item

Property Type Default Description
value string "" The value of the item (required)
disabled boolean false Whether the item is disabled

hal-select-group

Groups related items together. Has role="group".

hal-select-label

A label for a group of items. Styled smaller and muted.

hal-select-separator

A horizontal divider between groups.

Accessibility

  • Trigger has role="combobox" with aria-haspopup="listbox"
  • Trigger has aria-expanded to indicate open state
  • Content has role="listbox"
  • Items have role="option" with aria-selected
  • Groups have role="group"
  • Full keyboard navigation with ArrowUp/ArrowDown
  • Enter/Space to open and select
  • Escape to close
  • Home/End to jump to first/last item
