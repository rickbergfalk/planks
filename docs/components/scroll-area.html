Usage Notes

  • Native scrollbars are hidden using CSS scrollbar-width: none
  • The viewport automatically wraps content in a container for proper sizing
  • For vertical scrolling, set a fixed height on the scroll area
  • For horizontal scrolling, set a fixed width and use whitespace-nowrap
  • Supports drag interaction on the scrollbar thumb
  • Click on the scrollbar track to jump to that position
