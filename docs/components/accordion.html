Accordion

A vertically stacked set of interactive headings that each reveal a section of content.

Basic Usage

Is it accessible? Yes. It adheres to the WAI-ARIA design pattern. Is it styled? Yes. It comes with default styles that matches the other components. Is it animated? Yes. It's animated by default. 
<hal-accordion collapsible class="w-full">
  <hal-accordion-item value="item-1">
    <hal-accordion-trigger>Is it accessible?</hal-accordion-trigger>
    <hal-accordion-content>
      Yes. It adheres to the WAI-ARIA design pattern.
    </hal-accordion-content>
  </hal-accordion-item>
  <hal-accordion-item value="item-2">
    <hal-accordion-trigger>Is it styled?</hal-accordion-trigger>
    <hal-accordion-content>
      Yes. It comes with default styles.
    </hal-accordion-content>
  </hal-accordion-item>
</hal-accordion>

Default Open Item

First Item (Open) This item is open by default using the value attribute. Second Item Click to open this item. 
<hal-accordion value="item-1">
  <hal-accordion-item value="item-1">
    ...
  </hal-accordion-item>
</hal-accordion>

Multiple Mode

Set type="multiple" to allow multiple items to be open at once.

Item One Multiple items can be open at the same time. Item Two Try opening both items at once. 
<hal-accordion type="multiple">
  ...
</hal-accordion>

Collapsible

Set collapsible to allow closing all items in single mode.

Collapsible Item Click the trigger again to collapse this item. 
<hal-accordion collapsible>
  ...
</hal-accordion>

Disabled

Disabled Accordion This content cannot be revealed. 
<hal-accordion disabled>
  ...
</hal-accordion>

API

hal-accordion

Property Type Default Description
type "single" | "multiple" "single" Whether one or multiple items can be open at once
collapsible boolean false Whether items can be closed in single mode
value string "" The value(s) of open items (space-separated for multiple)
disabled boolean false Prevents all interaction

Events

Event Detail Description
value-change { value: string } Fired when the value changes

hal-accordion-item

Property Type Default Description
value string "" Unique identifier for the item
disabled boolean false Prevents interaction with this item

hal-accordion-trigger

The button that toggles the accordion item. Automatically sets up ARIA attributes (role="button", aria-expanded, aria-controls) and includes a chevron icon.

hal-accordion-content

The collapsible content. Hidden when closed, visible when open. Automatically gets an id for aria-controls.

Accessibility

  • Trigger has role="button" with aria-expanded state
  • Trigger has aria-controls pointing to the content's auto-generated id
  • Supports keyboard activation (Space and Enter keys)
  • Content uses hidden attribute when collapsed
  • Each accordion item has data-state for styling hooks
