Components

Component Description
hal-empty The empty state container
hal-empty-header Contains the icon, title, and description
hal-empty-media The icon/media container (variant: "default" | "icon")
hal-empty-title The empty state title
hal-empty-description The empty state description
hal-empty-content The content area (usually for actions)

Attributes

Component Attribute Values Default
hal-empty-media variant "default" | "icon" "default"
