Table

A responsive table with support for headers, body, footer, and captions.

React vs Web Component

Basic Usage

A simple table with header and body rows.

Invoice Status Method Amount INV001 Paid Credit Card $250.00 INV002 Pending PayPal $150.00 INV003 Unpaid Bank Transfer $350.00 
<hal-table>
  <hal-table-header>
    <hal-table-row>
      <hal-table-head>Invoice</hal-table-head>
      <hal-table-head>Status</hal-table-head>
      <hal-table-head class="text-right">Amount</hal-table-head>
    </hal-table-row>
  </hal-table-header>
  <hal-table-body>
    <hal-table-row>
      <hal-table-cell>INV001</hal-table-cell>
      <hal-table-cell>Paid</hal-table-cell>
      <hal-table-cell class="text-right">$250.00</hal-table-cell>
    </hal-table-row>
  </hal-table-body>
</hal-table>

With Caption

Add a caption to describe the table content.

A list of your recent invoices. Invoice Status Amount INV001 Paid $250.00 INV002 Pending $150.00 
<hal-table>
  <hal-table-caption>A list of your recent invoices.</hal-table-caption>
  <hal-table-header>...</hal-table-header>
  <hal-table-body>...</hal-table-body>
</hal-table>

With Footer

Add a footer for summary rows like totals.

Invoice Status Amount INV001 Paid $250.00 INV002 Pending $150.00 Total $400.00 
<hal-table-footer>
  <hal-table-row>
    <hal-table-cell colspan="2">Total</hal-table-cell>
    <hal-table-cell class="text-right">$400.00</hal-table-cell>
  </hal-table-row>
</hal-table-footer>

Selected Rows

Rows can be marked as selected for visual highlighting.

Invoice Status Amount INV001 Paid $250.00 INV002 Pending $150.00 INV003 Unpaid $350.00 
<hal-table-row selected>
  <hal-table-cell>INV002</hal-table-cell>
  ...
</hal-table-row>

Components

Component Description hal-table The table container with scrollable wrapper hal-table-header Contains header rows (thead) hal-table-body Contains data rows (tbody) hal-table-footer Contains footer/summary rows (tfoot) hal-table-row A table row with hover and selected states hal-table-head A header cell (th) with optional colspan, rowspan, scope hal-table-cell A data cell (td) with optional colspan, rowspan hal-table-caption A table caption element

hal-table-row

Property Type Default Description selected boolean false Whether the row is visually selected

hal-table-head / hal-table-cell

Property Type Default Description colspan number - Number of columns the cell spans rowspan number - Number of rows the cell spans scope (th only) string - Scope of the header cell (col, row, etc.)

Accessibility

  • Uses native semantic table elements (table, thead, tbody, tfoot, tr, th, td, caption)
  • Header cells support the scope attribute for screen readers
  • Selected rows use data-state="selected" for visual indication
  • Caption element provides a description of the table content
  • Supports colspan and rowspan for complex table structures
