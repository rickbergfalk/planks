Planks / Table

Table

A responsive table with support for headers, body, footer, and captions.

Basic Usage

A simple table with header and body rows.

Invoice Status Method Amount INV001 Paid Credit Card $250.00 INV002 Pending PayPal $150.00 INV003 Unpaid Bank Transfer $350.00 
<plank-table>
  <plank-table-header>
    <plank-table-row>
      <plank-table-head>Invoice</plank-table-head>
      <plank-table-head>Status</plank-table-head>
      <plank-table-head class="text-right">Amount</plank-table-head>
    </plank-table-row>
  </plank-table-header>
  <plank-table-body>
    <plank-table-row>
      <plank-table-cell>INV001</plank-table-cell>
      <plank-table-cell>Paid</plank-table-cell>
      <plank-table-cell class="text-right">$250.00</plank-table-cell>
    </plank-table-row>
  </plank-table-body>
</plank-table>

With Caption

Add a caption to describe the table content.

A list of your recent invoices. Invoice Status Amount INV001 Paid $250.00 INV002 Pending $150.00 
<plank-table>
  <plank-table-caption>A list of your recent invoices.</plank-table-caption>
  <plank-table-header>...</plank-table-header>
  <plank-table-body>...</plank-table-body>
</plank-table>

With Footer

Add a footer for summary rows like totals.

Invoice Status Amount INV001 Paid $250.00 INV002 Pending $150.00 Total $400.00 
<plank-table-footer>
  <plank-table-row>
    <plank-table-cell colspan="2">Total</plank-table-cell>
    <plank-table-cell class="text-right">$400.00</plank-table-cell>
  </plank-table-row>
</plank-table-footer>

Selected Rows

Rows can be marked as selected for visual highlighting.

Invoice Status Amount INV001 Paid $250.00 INV002 Pending $150.00 INV003 Unpaid $350.00 
<plank-table-row selected>
  <plank-table-cell>INV002</plank-table-cell>
  ...
</plank-table-row>

Components

Component Description plank-table The table container with scrollable wrapper plank-table-header Contains header rows (thead) plank-table-body Contains data rows (tbody) plank-table-footer Contains footer/summary rows (tfoot) plank-table-row A table row with hover and selected states plank-table-head A header cell (th) with optional colspan, rowspan, scope plank-table-cell A data cell (td) with optional colspan, rowspan plank-table-caption A table caption element

plank-table-row

Property Type Default Description selected boolean false Whether the row is visually selected

plank-table-head / plank-table-cell

Property Type Default Description colspan number - Number of columns the cell spans rowspan number - Number of rows the cell spans scope (th only) string - Scope of the header cell (col, row, etc.)

Accessibility

  • Uses native semantic table elements (table, thead, tbody, tfoot, tr, th, td, caption)
  • Header cells support the scope attribute for screen readers
  • Selected rows use data-state="selected" for visual indication
  • Caption element provides a description of the table content
  • Supports colspan and rowspan for complex table structures
