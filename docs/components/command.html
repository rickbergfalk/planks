Command

A command palette for searching and executing actions with keyboard navigation.

Basic Usage

A simple command palette with filterable items.

No results found. Calendar Search Emoji Calculator 
<plank-command class="rounded-lg border shadow-md">
  <plank-command-input placeholder="Type a command..."></plank-command-input>
  <plank-command-list>
    <plank-command-empty>No results found.</plank-command-empty>
    <plank-command-group heading="Suggestions">
      <plank-command-item>Calendar</plank-command-item>
      <plank-command-item>Search Emoji</plank-command-item>
    </plank-command-group>
  </plank-command-list>
</plank-command>

With Groups

Organize commands into logical groups with headings and separators.

No results found. Calendar Search Emoji Profile Billing Settings 
<plank-command-group heading="Suggestions">
  <plank-command-item>Calendar</plank-command-item>
</plank-command-group>
<plank-command-separator></plank-command-separator>
<plank-command-group heading="Settings">
  <plank-command-item>Profile</plank-command-item>
</plank-command-group>

With Shortcuts

Display keyboard shortcuts alongside command items.

Profile ⌘P Billing ⌘B Settings ⌘S 
<plank-command-item>
  <span>Profile</span>
  <plank-command-shortcut>⌘P</plank-command-shortcut>
</plank-command-item>

Disabled Items

Individual items can be disabled and will be skipped during keyboard navigation.

Enabled Action Disabled Action Another Action 
<plank-command-item disabled>Disabled Action</plank-command-item>

API

plank-command

The root container that manages state and filtering.

Events

Event Detail Description
select { value: string } Fired when an item is selected

plank-command-input

Property Type Default Description
placeholder string "" Placeholder text for the input

plank-command-list

Container for items. Has role="listbox" and scrollable overflow.

plank-command-empty

Displayed when no items match the search query.

plank-command-group

Property Type Default Description
heading string "" The heading text for the group

plank-command-item

Property Type Default Description
value string "" The value (defaults to lowercase text content)
disabled boolean false Whether the item is disabled

plank-command-separator

A horizontal divider between groups.

plank-command-shortcut

Displays a keyboard shortcut aligned to the right of an item.

Accessibility

  • Input has role="combobox" with aria-expanded="true"
  • Input has aria-controls pointing to the list
  • List has role="listbox"
  • Items have role="option" with aria-selected
  • Groups have role="group"
  • Separators have role="separator"
  • Full keyboard navigation with ArrowUp/ArrowDown
  • Enter to select the highlighted item
  • Home/End to jump to first/last item
  • Disabled items are skipped during navigation
