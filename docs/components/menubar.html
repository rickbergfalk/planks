Menubar

A visually persistent menu common in desktop applications that provides quick access to a consistent set of commands.

Basic Usage

File New Tab ⌘T New Window ⌘N Share Print ⌘P Edit Undo ⌘Z Redo ⇧⌘Z Cut ⌘X Copy ⌘C Paste ⌘V View Status Bar Activity Bar Zoom In ⌘+ Zoom Out ⌘- 
<plank-menubar>
  <plank-menubar-menu>
    <plank-menubar-trigger>File</plank-menubar-trigger>
    <plank-menubar-content>
      <plank-menubar-item>
        New Tab
        <plank-menubar-shortcut>⌘T</plank-menubar-shortcut>
      </plank-menubar-item>
      <plank-menubar-separator></plank-menubar-separator>
      <plank-menubar-item>Print</plank-menubar-item>
    </plank-menubar-content>
  </plank-menubar-menu>
</plank-menubar>

Use checkbox items for toggleable options.

</plank-menubar-checkbox-item>

Radio Group

Use radio groups for mutually exclusive options.

  <plank-menubar-radio-item value="top">Top</plank-menubar-radio-item>
  <plank-menubar-radio-item value="bottom">Bottom</plank-menubar-radio-item>
  <plank-menubar-radio-item value="right">Right</plank-menubar-radio-item>
</plank-menubar-radio-group>

Disabled Items

Use the disabled attribute to disable menu items.

<plank-menubar-item disabled>Delete</plank-menubar-item>

API

plank-menubar

The root container for the menubar. Has role="menubar".

plank-menubar-menu

Container for a single menu within the menubar.

plank-menubar-trigger

The button that opens the menu. Has role="menuitem".

plank-menubar-content

Property Type Default Description
side top | right | bottom | left bottom The preferred side
align start | center | end start The preferred alignment
sideOffset number 8 Offset from the trigger in pixels

plank-menubar-item

Property Type Default Description
disabled boolean false Whether the item is disabled
variant default | destructive default The visual variant
inset boolean false Add left padding for alignment

plank-menubar-checkbox-item

Property Type Default Description
checked boolean false Whether the checkbox is checked
disabled boolean false Whether the item is disabled

Events

Event Detail Description
open-change { open: boolean, menu: Element } Fired when a menu opens or closes
select { item: Element } Fired when a menu item is selected
checked-change { item: Element, checked: boolean } Fired when a checkbox item changes
value-change { value: string } Fired when a radio group value changes

Accessibility

  • Menubar has role="menubar"
  • Triggers have role="menuitem" and aria-haspopup="menu"
  • Menu content has role="menu"
  • Menu items have role="menuitem"
  • Checkbox items have role="menuitemcheckbox" with aria-checked
  • Radio items have role="menuitemradio" with aria-checked
  • Separators have role="separator"
  • Disabled items have aria-disabled="true"
  • Arrow key navigation between menus
  • Closes on Escape key
  • Closes when clicking outside

Keyboard Navigation

Key Action
Enter / Space / ArrowDown Open menu (when trigger focused)
ArrowLeft / ArrowRight Navigate between menu triggers
ArrowDown Move to next item (in menu content)
ArrowUp Move to previous item (in menu content)
Home Move to first item
End Move to last item
Enter / Space Select focused item
Escape Close menu
