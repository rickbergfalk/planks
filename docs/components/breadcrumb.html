Breadcrumb

Displays the path to the current resource using a hierarchy of links.

React vs Web Component

Basic Usage

A simple breadcrumb navigation with links and current page indicator.

Home Components Breadcrumb 
<hal-breadcrumb>
  <hal-breadcrumb-list>
    <hal-breadcrumb-item>
      <hal-breadcrumb-link href="#">Home</hal-breadcrumb-link>
    </hal-breadcrumb-item>
    <hal-breadcrumb-separator></hal-breadcrumb-separator>
    <hal-breadcrumb-item>
      <hal-breadcrumb-link href="#">Components</hal-breadcrumb-link>
    </hal-breadcrumb-item>
    <hal-breadcrumb-separator></hal-breadcrumb-separator>
    <hal-breadcrumb-item>
      <hal-breadcrumb-page>Breadcrumb</hal-breadcrumb-page>
    </hal-breadcrumb-item>
  </hal-breadcrumb-list>
</hal-breadcrumb>

Components

Component Description hal-breadcrumb The navigation container with aria-label="breadcrumb" hal-breadcrumb-list Container for breadcrumb items with flex layout hal-breadcrumb-item Wrapper for individual breadcrumb elements hal-breadcrumb-link A navigable link to a page hal-breadcrumb-page The current page (not a link) hal-breadcrumb-separator Chevron separator between items hal-breadcrumb-ellipsis Ellipsis indicator for collapsed items

hal-breadcrumb-link

Property Type Default Description href string "" The URL for the link

Accessibility

  • Uses aria-label="breadcrumb" on the nav container
  • Current page has aria-current="page"
  • Separators and ellipsis have aria-hidden="true"
  • Ellipsis includes screen-reader fallback text
