Breadcrumb

Displays the path to the current resource using a hierarchy of links.

Basic Usage

A simple breadcrumb navigation with links and current page indicator.

Home Components Breadcrumb 
<plank-breadcrumb>
  <plank-breadcrumb-list>
    <plank-breadcrumb-item>
      <plank-breadcrumb-link href="#">Home</plank-breadcrumb-link>
    </plank-breadcrumb-item>
    <plank-breadcrumb-separator></plank-breadcrumb-separator>
    <plank-breadcrumb-item>
      <plank-breadcrumb-link href="#">Components</plank-breadcrumb-link>
    </plank-breadcrumb-item>
    <plank-breadcrumb-separator></plank-breadcrumb-separator>
    <plank-breadcrumb-item>
      <plank-breadcrumb-page>Breadcrumb</plank-breadcrumb-page>
    </plank-breadcrumb-item>
  </plank-breadcrumb-list>
</plank-breadcrumb>

With Ellipsis

Use the ellipsis component to indicate collapsed breadcrumb items.

</plank-breadcrumb>

Components

Component Description plank-breadcrumb The navigation container with aria-label="breadcrumb" plank-breadcrumb-list Container for breadcrumb items with flex layout plank-breadcrumb-item Wrapper for individual breadcrumb elements plank-breadcrumb-link A navigable link to a page plank-breadcrumb-page The current page (not a link) plank-breadcrumb-separator Chevron separator between items plank-breadcrumb-ellipsis Ellipsis indicator for collapsed items

plank-breadcrumb-link

Property Type Default Description href string "" The URL for the link

Accessibility

  • Uses aria-label="breadcrumb" on the nav container
  • Current page has aria-current="page"
  • Separators and ellipsis have aria-hidden="true"
  • Ellipsis includes screen-reader fallback text
