API

hal-sidebar-provider

Root component that provides sidebar state context.

Attribute Type Default Description
default-open boolean true Initial open state
open boolean - Controlled open state

hal-sidebar

Main sidebar container.

Attribute Type Default Description
side "left" | "right" "left" Which side to show sidebar
variant "sidebar" | "floating" | "inset" "sidebar" Visual variant
collapsible "offcanvas" | "icon" | "none" "offcanvas" Collapse behavior

hal-sidebar-menu-button

Button within a menu item.

Attribute Type Default Description
variant "default" | "outline" "default" Visual variant
size "default" | "sm" | "lg" "default" Button size
active boolean false Active state
tooltip string - Tooltip when collapsed
