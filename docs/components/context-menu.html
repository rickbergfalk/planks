Context Menu

Displays a menu to the user triggered by a right-click (context menu event).

Basic Usage

Right-click on the trigger area to open the context menu.

Right click here
My Account Profile ⇧⌘P Billing ⌘B Settings ⌘S Log out 
<plank-context-menu>
  <plank-context-menu-trigger>
    <div class="trigger-area">Right click here</div>
  </plank-context-menu-trigger>
  <plank-context-menu-content>
    <plank-context-menu-label>My Account</plank-context-menu-label>
    <plank-context-menu-separator></plank-context-menu-separator>
    <plank-context-menu-group>
      <plank-context-menu-item>
        Profile
        <plank-context-menu-shortcut>⇧⌘P</plank-context-menu-shortcut>
      </plank-context-menu-item>
      <plank-context-menu-item>
        Billing
        <plank-context-menu-shortcut>⌘B</plank-context-menu-shortcut>
      </plank-context-menu-item>
    </plank-context-menu-group>
    <plank-context-menu-separator></plank-context-menu-separator>
    <plank-context-menu-item>Log out</plank-context-menu-item>
  </plank-context-menu-content>
</plank-context-menu>

Checkbox Items

Use checkbox items for toggleable options.

Right click here
Appearance Status Bar Activity Bar Panel 
<plank-context-menu-checkbox-item checked>
  Status Bar
</plank-context-menu-checkbox-item>

Radio Group

Use radio groups for mutually exclusive options.

Right click here
Panel Position Top Bottom Right 
<plank-context-menu-radio-group value="bottom">
  <plank-context-menu-radio-item value="top">Top</plank-context-menu-radio-item>
  <plank-context-menu-radio-item value="bottom">Bottom</plank-context-menu-radio-item>
  <plank-context-menu-radio-item value="right">Right</plank-context-menu-radio-item>
</plank-context-menu-radio-group>

Disabled Items

Use the disabled attribute to disable menu items.

Right click here
Edit Delete Duplicate 
<plank-context-menu-item disabled>Delete</plank-context-menu-item>

Destructive Items

Use variant="destructive" for dangerous actions.

Right click here
Edit Duplicate Delete 
<plank-context-menu-item variant="destructive">Delete</plank-context-menu-item>

Inset Items

Use the inset attribute to align items without icons.

Right click here
Settings Profile Account 
<plank-context-menu-label inset>Settings</plank-context-menu-label>
<plank-context-menu-item inset>Profile</plank-context-menu-item>

Submenus

Use nested submenus for hierarchical navigation.

Right click here
New Tab New Window More Tools Save Page As... Create Shortcut... Name Window... Developer Tools Quit 
<plank-context-menu-sub>
  <plank-context-menu-sub-trigger>More Tools</plank-context-menu-sub-trigger>
  <plank-context-menu-sub-content>
    <plank-context-menu-item>Save Page As...</plank-context-menu-item>
    <plank-context-menu-item>Developer Tools</plank-context-menu-item>
  </plank-context-menu-sub-content>
</plank-context-menu-sub>

API

plank-context-menu

Property Type Default Description
open boolean false The controlled open state

Events

Event Detail Description
open-change { open: boolean } Fired when the open state changes
select { item: Element } Fired when a menu item is selected
checked-change { item: Element, checked: boolean } Fired when a checkbox item changes
value-change { value: string } Fired when a radio group value changes

plank-context-menu-content

Property Type Default Description
alignOffset number -4 Offset from the cursor position in pixels (align axis)

plank-context-menu-item

Property Type Default Description
disabled boolean false Whether the item is disabled
variant default | destructive default The visual variant
inset boolean false Add left padding for alignment

plank-context-menu-checkbox-item

Property Type Default Description
checked boolean false Whether the checkbox is checked
disabled boolean false Whether the item is disabled

plank-context-menu-radio-group

Property Type Default Description
value string "" The selected value

plank-context-menu-radio-item

Property Type Default Description
value string "" The value for this item
disabled boolean false Whether the item is disabled

plank-context-menu-label

Property Type Default Description
inset boolean false Add left padding for alignment

Accessibility

  • Menu content has role="menu"
  • Menu items have role="menuitem"
  • Checkbox items have role="menuitemcheckbox" with aria-checked
  • Radio items have role="menuitemradio" with aria-checked
  • Separators have role="separator"
  • Disabled items have aria-disabled="true"
  • Arrow key navigation between items
  • Closes on Escape key
  • Closes when clicking outside

Keyboard Navigation

Key Action
ArrowDown Move to next item
ArrowUp Move to previous item
Home Move to first item
End Move to last item
Enter / Space Select focused item
Escape Close menu

Comparison to Dropdown Menu

Context Menu and Dropdown Menu share the same structure and styling, but differ in their trigger mechanism:

  • Dropdown Menu: Triggered by clicking (left-click) on a button. Menu appears anchored to the trigger element.
  • Context Menu: Triggered by right-clicking on a region. Menu appears at the cursor position.
