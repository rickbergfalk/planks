Planks / Pagination

Pagination

Pagination with page navigation, previous/next buttons, and ellipsis.

Basic Usage

A simple pagination with previous/next and page links.

1 2 3 
<plank-pagination>
  <plank-pagination-content>
    <plank-pagination-item>
      <plank-pagination-previous href="#"></plank-pagination-previous>
    </plank-pagination-item>
    <plank-pagination-item>
      <plank-pagination-link href="#">1</plank-pagination-link>
    </plank-pagination-item>
    <plank-pagination-item>
      <plank-pagination-link href="#" active>2</plank-pagination-link>
    </plank-pagination-item>
    <plank-pagination-item>
      <plank-pagination-ellipsis></plank-pagination-ellipsis>
    </plank-pagination-item>
    <plank-pagination-item>
      <plank-pagination-next href="#"></plank-pagination-next>
    </plank-pagination-item>
  </plank-pagination-content>
</plank-pagination>

Components

Component Description plank-pagination The navigation container with role="navigation" plank-pagination-content Container for the pagination items plank-pagination-item Wrapper for individual pagination elements plank-pagination-link A page number link plank-pagination-previous Previous page link plank-pagination-next Next page link plank-pagination-ellipsis Ellipsis indicator for hidden pages

plank-pagination-link

Property Type Default Description href string "" The URL for the link active boolean false Whether this is the current page size "default" | "icon" "icon" The size of the link button

Accessibility

  • Uses role="navigation" with aria-label="pagination"
  • Active page has aria-current="page"
  • Previous/Next links have descriptive aria-label
  • Ellipsis has aria-hidden with screen-reader fallback text
