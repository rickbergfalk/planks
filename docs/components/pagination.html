Pagination

Pagination with page navigation, previous/next buttons, and ellipsis.

React vs Web Component

Basic Usage

A simple pagination with previous/next and page links.

1 2 3 
<hal-pagination>
  <hal-pagination-content>
    <hal-pagination-item>
      <hal-pagination-previous href="#"></hal-pagination-previous>
    </hal-pagination-item>
    <hal-pagination-item>
      <hal-pagination-link href="#">1</hal-pagination-link>
    </hal-pagination-item>
    <hal-pagination-item>
      <hal-pagination-link href="#" active>2</hal-pagination-link>
    </hal-pagination-item>
    <hal-pagination-item>
      <hal-pagination-ellipsis></hal-pagination-ellipsis>
    </hal-pagination-item>
    <hal-pagination-item>
      <hal-pagination-next href="#"></hal-pagination-next>
    </hal-pagination-item>
  </hal-pagination-content>
</hal-pagination>

Components

Component Description hal-pagination The navigation container with role="navigation" hal-pagination-content Container for the pagination items hal-pagination-item Wrapper for individual pagination elements hal-pagination-link A page number link hal-pagination-previous Previous page link hal-pagination-next Next page link hal-pagination-ellipsis Ellipsis indicator for hidden pages

hal-pagination-link

Property Type Default Description href string "" The URL for the link active boolean false Whether this is the current page size "default" | "icon" "icon" The size of the link button

Accessibility

  • Uses role="navigation" with aria-label="pagination"
  • Active page has aria-current="page"
  • Previous/Next links have descriptive aria-label
  • Ellipsis has aria-hidden with screen-reader fallback text
