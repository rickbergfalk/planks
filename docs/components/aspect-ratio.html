Aspect Ratio

Maintains a desired aspect ratio for content.

16:9 Widescreen

16:9 
<hal-aspect-ratio ratio="16/9" class="bg-muted rounded-lg">
  <!-- content -->
</hal-aspect-ratio>

4:3 Standard

4:3 
<hal-aspect-ratio ratio="4/3" class="bg-muted rounded-lg">
  <!-- content -->
</hal-aspect-ratio>

1:1 Square

1:1 
<hal-aspect-ratio ratio="1/1" class="bg-muted rounded-lg">
  <!-- content -->
</hal-aspect-ratio>

With Image

<hal-aspect-ratio ratio="16/9" class="rounded-lg">
  <img src="..." class="h-full w-full object-cover" />
</hal-aspect-ratio>

Props

Prop Type Default Description
ratio string "1/1" Aspect ratio in "width/height" format (e.g., "16/9", "4/3")
