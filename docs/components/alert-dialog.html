Alert Dialog

A modal dialog that interrupts the user with important content and expects a response. Unlike regular dialogs, alert dialogs cannot be dismissed by clicking the overlay.

Basic Usage

Delete Account Are you absolutely sure? This action cannot be undone. This will permanently delete your account and remove your data from our servers. Cancel Continue 
<hal-alert-dialog>
  <hal-alert-dialog-trigger>
    <hal-button variant="destructive">Delete Account</hal-button>
  </hal-alert-dialog-trigger>
  <hal-alert-dialog-content>
    <hal-alert-dialog-header>
      <hal-alert-dialog-title>Are you absolutely sure?</hal-alert-dialog-title>
      <hal-alert-dialog-description>
        This action cannot be undone. This will permanently delete
        your account and remove your data from our servers.
      </hal-alert-dialog-description>
    </hal-alert-dialog-header>
    <hal-alert-dialog-footer>
      <hal-alert-dialog-cancel>Cancel</hal-alert-dialog-cancel>
      <hal-alert-dialog-action>Continue</hal-alert-dialog-action>
    </hal-alert-dialog-footer>
  </hal-alert-dialog-content>
</hal-alert-dialog>

Simple Confirmation

A simpler alert dialog for quick confirmations.

Delete Item Delete Item Are you sure you want to delete this item? No Yes 
<hal-alert-dialog>
  <hal-alert-dialog-trigger>
    <hal-button variant="outline">Delete Item</hal-button>
  </hal-alert-dialog-trigger>
  <hal-alert-dialog-content>
    <hal-alert-dialog-header>
      <hal-alert-dialog-title>Delete Item</hal-alert-dialog-title>
      <hal-alert-dialog-description>
        Are you sure you want to delete this item?
      </hal-alert-dialog-description>
    </hal-alert-dialog-header>
    <hal-alert-dialog-footer>
      <hal-alert-dialog-cancel>No</hal-alert-dialog-cancel>
      <hal-alert-dialog-action>Yes</hal-alert-dialog-action>
    </hal-alert-dialog-footer>
  </hal-alert-dialog-content>
</hal-alert-dialog>

API

hal-alert-dialog

Property Type Default Description
open boolean false The controlled open state

Events

Event Detail Description
open-change { open: boolean } Fired when the open state changes

hal-alert-dialog-trigger

The element that triggers the alert dialog. Opens dialog on click.

hal-alert-dialog-content

Property Type Default Description
class string "" Additional CSS classes

hal-alert-dialog-header

Container for title and description. Provides proper spacing.

hal-alert-dialog-footer

Container for action buttons. Stacks on mobile, row on desktop.

hal-alert-dialog-title

The dialog title. Used for aria-labelledby.

hal-alert-dialog-description

The dialog description. Used for aria-describedby.

hal-alert-dialog-action

The confirm/action button. Has primary button styling. Closes the dialog when clicked.

hal-alert-dialog-cancel

The cancel button. Has outline button styling. Closes the dialog when clicked.

Dialog vs Alert Dialog

Feature Dialog Alert Dialog
ARIA role dialog alertdialog
Click overlay to close Yes No
Close button Yes (default) No
Escape to close Yes Yes
Use case General content Confirmations, destructive actions

Accessibility

  • Dialog content has role="alertdialog"
  • Trigger has aria-haspopup="dialog"
  • Trigger has aria-expanded to indicate open state
  • Trigger has aria-controls pointing to the dialog when open
  • Dialog has aria-labelledby pointing to the title
  • Dialog has aria-describedby pointing to the description
  • Closes on Escape key
  • Does NOT close when clicking overlay (requires explicit action)
  • Focus is moved to Cancel button (or Action if no Cancel) when opened
