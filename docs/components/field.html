Field

A set of components for building accessible, well-structured form fields.

Basic Field

Email 
<hal-field>
  <hal-field-label>Email</hal-field-label>
  <hal-input type="email" placeholder="email@example.com"></hal-input>
</hal-field>

With Description

Username This will be your public display name. 
<hal-field>
  <hal-field-label>Username</hal-field-label>
  <hal-input placeholder="johndoe"></hal-input>
  <hal-field-description>
    This will be your public display name.
  </hal-field-description>
</hal-field>

With Error

Password Password must be at least 8 characters. 
<hal-field data-invalid="true">
  <hal-field-label>Password</hal-field-label>
  <hal-input type="password"></hal-input>
  <hal-field-error>
    Password must be at least 8 characters.
  </hal-field-error>
</hal-field>

Components

Component Description
hal-field Individual field wrapper with orientation support
hal-field-set Groups related fields together
hal-field-legend Title for a fieldset
hal-field-group Container for grouping multiple fields
hal-field-content Content area within a field
hal-field-label Label for a field input
hal-field-title Non-label title for display
hal-field-description Help text for a field
hal-field-separator Visual divider with optional text
hal-field-error Error message display

Attributes

Component Attribute Values Default
hal-field orientation "vertical" | "horizontal" | "responsive" "vertical"
hal-field data-invalid "true" | "false" -
hal-field-legend variant "legend" | "label" "legend"
hal-field-label for string (input ID) -
