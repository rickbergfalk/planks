Planks / Chart

Chart

Chart container and helper components for building data visualizations.

Note: Unlike the React version which uses Recharts, the web component version is library-agnostic. Use any charting library (Chart.js, D3, ApexCharts, etc.) inside the container and reference the CSS custom properties for colors.

Chart Container

The chart container provides CSS custom properties for chart colors based on your config. 

<plank-chart-container chart-id="demo-chart">
  <!-- Your chart content here -->
  <!-- Colors available via CSS: var(--color-desktop), var(--color-mobile) -->
</plank-chart-container>

<script>
  const chart = document.querySelector('plank-chart-container');
  chart.config = {
    desktop: { label: "Desktop", color: "#2563eb" },
    mobile: { label: "Mobile", color: "#60a5fa" }
  };
</script>

Chart Tooltip

Tooltip component for displaying chart data on hover. 

<plank-chart-tooltip active label="January"></plank-chart-tooltip>

<script>
  const tooltip = document.querySelector('plank-chart-tooltip');
  tooltip.items = [
    { name: "Desktop", value: 186, color: "#2563eb" },
    { name: "Mobile", value: 80, color: "#60a5fa" }
  ];
</script>

Chart Legend

Legend component for chart data series. 

<plank-chart-legend></plank-chart-legend>

<script>
  const legend = document.querySelector('plank-chart-legend');
  legend.items = [
    { name: "Desktop", color: "#2563eb" },
    { name: "Mobile", color: "#60a5fa" },
    { name: "Tablet", color: "#8b5cf6" }
  ];
</script>

API

plank-chart-container

Property Type Default Description
chart-id string auto Unique ID for the chart (auto-generated if not provided)
config ChartConfig {} Configuration object defining labels and colors for data series

ChartConfig Type

type ChartConfig = {
  [key: string]: {
    label?: string
    color?: string
    theme?: { light: string; dark: string }
  }
}

plank-chart-tooltip

Property Type Default Description
active boolean false Whether tooltip is visible
label string - Label text (e.g., category name)
items Array [] Array of { name, value, color } objects
indicator "dot" | "line" | "dashed" "dot" Style of color indicator
hide-label boolean false Hide the label
hide-indicator boolean false Hide color indicators

plank-chart-legend

Property Type Default Description
items Array [] Array of { name, color } objects
vertical-align "top" | "bottom" "bottom" Vertical alignment (affects padding)
hide-icon boolean false Hide color icons
