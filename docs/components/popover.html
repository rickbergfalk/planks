Popover

Displays rich content in a portal, triggered by a button.

Basic Usage

Open popover

Dimensions

Set the dimensions for the layer. 

<plank-popover>
  <plank-popover-trigger>
    <plank-button variant="outline">Open popover</plank-button>
  </plank-popover-trigger>
  <plank-popover-content>
    <div class="grid gap-4">
      <div class="space-y-2">
        <h4 class="leading-none font-medium">Dimensions</h4>
        <p class="text-muted-foreground text-sm">
          Set the dimensions for the layer.
        </p>
      </div>
    </div>
  </plank-popover-content>
</plank-popover>

Alignment

Use the align attribute to control how the popover aligns with the trigger.

Start

Aligned to start

 Center

Aligned to center

 End

Aligned to end

 
<plank-popover-content align="start">...</plank-popover-content>
<plank-popover-content align="center">...</plank-popover-content>
<plank-popover-content align="end">...</plank-popover-content>

Different Sides

Top

Popover on top

 Right

Popover on right

 Bottom

Popover on bottom

 Left

Popover on left

 
<plank-popover-content side="top">...</plank-popover-content>

<plank-popover-content side="bottom">...</plank-popover-content>

Controlled State

Use the open attribute to control the popover state programmatically.

Always open

This popover is controlled

 
<plank-popover open>
  ...
</plank-popover>

API

plank-popover

Property Type Default Description
open boolean false The controlled open state

Events

Event Detail Description
open-change { open: boolean } Fired when the open state changes

plank-popover-trigger

The element that triggers the popover. Opens popover on click.

plank-popover-content

Property Type Default Description
side "top" | "right" | "bottom" | "left" "bottom" The preferred side for the popover
align "start" | "center" | "end" "center" The alignment along the side
side-offset number 4 Distance from the trigger in pixels

Accessibility

  • Popover content has role="dialog"
  • Trigger has aria-haspopup="dialog"
  • Trigger has aria-expanded to indicate open state
  • Trigger has aria-controls pointing to the popover when open
  • Closes on Escape key
  • Closes when clicking outside
  • Uses Floating UI for smart positioning that avoids viewport edges
