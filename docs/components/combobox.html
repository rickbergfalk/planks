Combobox

A searchable dropdown for selecting from a list of options.

Basic Usage

A simple combobox with searchable items.

Next.js SvelteKit Nuxt.js Remix Astro 
<plank-combobox placeholder="Select framework...">
  <plank-combobox-item value="next">Next.js</plank-combobox-item>
  <plank-combobox-item value="svelte">SvelteKit</plank-combobox-item>
  <plank-combobox-item value="nuxt">Nuxt.js</plank-combobox-item>
</plank-combobox>

Preselected Value

Set an initial value using the value attribute.

Next.js SvelteKit Nuxt.js 
<plank-combobox value="svelte" placeholder="Select framework...">
  <plank-combobox-item value="next">Next.js</plank-combobox-item>
  <plank-combobox-item value="svelte">SvelteKit</plank-combobox-item>
  <plank-combobox-item value="nuxt">Nuxt.js</plank-combobox-item>
</plank-combobox>

Disabled Items

Individual items can be disabled and will be skipped during keyboard navigation.

Next.js SvelteKit (Unavailable) Nuxt.js 
<plank-combobox-item value="svelte" disabled>
  SvelteKit (Unavailable)
</plank-combobox-item>

Disabled Combobox

The entire combobox can be disabled.

Next.js 
<plank-combobox disabled placeholder="Select framework...">
  <plank-combobox-item value="next">Next.js</plank-combobox-item>
</plank-combobox>

Custom Empty Text

Customize the message shown when no items match the search query.

Next.js SvelteKit 
<plank-combobox
  placeholder="Select framework..."
  emptyText="No frameworks found. Try another search."
>

API

plank-combobox

Property Type Default Description
value string "" The currently selected value
open boolean false Whether the dropdown is open
placeholder string "Select..." Placeholder text when no value selected
searchPlaceholder string "Search..." Placeholder for the search input
emptyText string "No results found." Message shown when search finds no matches
disabled boolean false Whether the combobox is disabled

Events

Event Detail Description
value-change { value: string } Fired when the selected value changes
open-change { open: boolean } Fired when the dropdown opens or closes

plank-combobox-item

Property Type Default Description
value string "" The value of the item
disabled boolean false Whether the item is disabled

Accessibility

  • Trigger has role="combobox" with aria-expanded
  • Trigger has aria-haspopup="listbox"
  • Trigger has aria-controls pointing to the listbox
  • Dropdown list has role="listbox"
  • Items have role="option" with aria-selected
  • Disabled items have aria-disabled="true"
  • Full keyboard navigation with ArrowUp/ArrowDown
  • Enter to select the highlighted item
  • Home/End to jump to first/last item
  • Escape to close the dropdown
  • Disabled items are skipped during navigation
  • Type to filter items
