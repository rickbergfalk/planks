Components

Component Description plank-resizable-panel-group Container for resizable panels plank-resizable-panel A resizable panel plank-resizable-handle Draggable handle between panels

plank-resizable-panel-group

Property Type Default Description direction "horizontal" | "vertical" "horizontal" Direction of panel arrangement

plank-resizable-panel

Property Type Default Description default-size number - Initial size as percentage (0-100) min-size number 0 Minimum size as percentage max-size number 100 Maximum size as percentage

plank-resizable-handle

Property Type Default Description with-handle boolean false Show visible grip icon

Events

Event Detail Description resize { sizes: number[] } Fired when panel sizes change

Usage Notes

  • Panel sizes are specified as percentages and must total 100%
  • Panels without default-size get remaining space distributed equally
  • Keyboard navigation: Arrow keys resize, Shift+Arrow for larger steps
  • The handle has role="separator" and proper ARIA attributes
  • Supports nested panel groups for complex layouts
