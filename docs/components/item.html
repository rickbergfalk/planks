Components

Component Description
hal-item The main item container
hal-item-group Groups multiple items with role="list"
hal-item-separator Visual separator between items
hal-item-media Container for icon or image (variant: "default" | "icon" | "image")
hal-item-content Container for title and description
hal-item-title The item title
hal-item-description The item description
hal-item-actions Container for action buttons
hal-item-header Header row (full width)
hal-item-footer Footer row (full width)

Attributes

Component Attribute Values Default
hal-item variant "default" | "outline" | "muted" "default"
hal-item size "default" | "sm" "default"
hal-item-media variant "default" | "icon" | "image" "default"
