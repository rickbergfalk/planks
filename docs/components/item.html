Components

Component Description
plank-item The main item container
plank-item-group Groups multiple items with role="list"
plank-item-separator Visual separator between items
plank-item-media Container for icon or image (variant: "default" | "icon" | "image")
plank-item-content Container for title and description
plank-item-title The item title
plank-item-description The item description
plank-item-actions Container for action buttons
plank-item-header Header row (full width)
plank-item-footer Footer row (full width)

Attributes

Component Attribute Values Default
plank-item variant "default" | "outline" | "muted" "default"
plank-item size "default" | "sm" "default"
plank-item-media variant "default" | "icon" | "image" "default"
