Input

Displays a form input field.

Default

<hal-input placeholder="Enter text..."></hal-input>

With Label

Email 
<hal-label for="email">Email</hal-label>
<hal-input id="email" type="email" placeholder="email@example.com"></hal-input>

Disabled

<hal-input placeholder="Disabled input" disabled></hal-input>

API

Property Type Default Description
type string "text" Input type (text, email, password, etc.)
placeholder string "" Placeholder text
value string "" Input value
disabled boolean false Disables the input
