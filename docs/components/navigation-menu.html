API

hal-navigation-menu

Property Type Default Description
viewport boolean true Whether to use a shared viewport container
delay-duration number 200 Delay in ms before opening on hover
skip-delay-duration number 300 Time window after closing where next open is instant

hal-navigation-menu-trigger

Property Type Default Description
disabled boolean false Whether the trigger is disabled

hal-navigation-menu-link

Property Type Default Description
href string "" The URL the link points to
active boolean false Whether the link is currently active

Events

Event Detail Description
value-change { value: string } Fired on hal-navigation-menu when the open item changes

Accessibility

  • Navigation list has role="menubar"
  • Menu items have role="menuitem"
  • Trigger has aria-haspopup="menu" and aria-expanded
  • Content has role="menu"
  • Links have role="link" inside menu content
  • Arrow key navigation between menu items
  • Opens on hover with configurable delay
  • Closes on Escape key
  • Closes when clicking outside

Keyboard Navigation

Key Action
Enter / Space Open/close menu (on trigger)
ArrowDown Open menu (on trigger)
ArrowLeft / ArrowRight Navigate between menu items
Escape Close menu and return focus to trigger
Tab Navigate within content, closes when leaving
