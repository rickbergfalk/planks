Radio Group

A set of checkable buttons where only one can be checked at a time.

Basic Usage

Default
Comfortable
Compact

API

plank-radio-group

Property Type Default Description
value string "" The value of the currently selected item
disabled boolean false Disables all items in the group

Events

Event Detail Description
value-change { value: string } Fired when the selected item changes

plank-radio-group-item

Property Type Default Description
value string "" Unique identifier for the item
disabled boolean false Disables this item

Accessibility

  • Radio group has role="radiogroup"
  • Each item has role="radio" with aria-checked state
  • Arrow keys (Up/Down/Left/Right) navigate and select between items
  • Space key selects the focused item
  • Navigation wraps from last to first item and vice versa
  • Only one item in the group is focusable at a time (roving tabindex)
  • Disabled items are skipped during keyboard navigation
