Toggle Group

A set of two-state buttons that can be toggled on or off.

React vs Web Component

Multiple Selection

Multiple items can be selected at once.

B I U 
<hal-toggle-group type="multiple" value="bold">
  <hal-toggle-group-item value="bold">B</hal-toggle-group-item>
  <hal-toggle-group-item value="italic">I</hal-toggle-group-item>
  <hal-toggle-group-item value="underline">U</hal-toggle-group-item>
</hal-toggle-group>

Single Selection

Only one item can be selected at a time.

L C R 
<hal-toggle-group type="single" value="center">
  <hal-toggle-group-item value="left">L</hal-toggle-group-item>
  <hal-toggle-group-item value="center">C</hal-toggle-group-item>
  <hal-toggle-group-item value="right">R</hal-toggle-group-item>
</hal-toggle-group>

API

hal-toggle-group

Property Type Default Description
type "single" | "multiple" "multiple" Selection mode
value string "" Selected value(s), comma-separated for multiple
variant "default" | "outline" "default" Visual variant
size "default" | "sm" | "lg" "default" Size of items
disabled boolean false Disables all items

Events

Event Detail Description
value-change { value: string } Fired when selection changes

hal-toggle-group-item

Property Type Default Description
value string "" Unique identifier for the item
disabled boolean false Disables this item

Accessibility

  • Toggle group has role="group"
  • Each item has role="button" with aria-pressed state
  • Space and Enter keys toggle the focused item
  • Items are focusable via Tab key navigation
  • Disabled items are removed from tab order
