Calendar

A date picker component with single, multiple, and range selection modes.

Basic Calendar

<hal-calendar class="rounded-lg border shadow-sm"></hal-calendar>

Single Date Selection

<hal-calendar
  mode="single"
  selected="2025-01-15"
  class="rounded-lg border shadow-sm"
></hal-calendar>

Multiple Date Selection

<hal-calendar
  mode="multiple"
  selected-dates="2025-01-10,2025-01-15,2025-01-20"
  class="rounded-lg border shadow-sm"
></hal-calendar>

Date Range Selection

<hal-calendar
  mode="range"
  range-start="2025-01-10"
  range-end="2025-01-20"
  class="rounded-lg border shadow-sm"
></hal-calendar>

Multiple Months

<hal-calendar
  number-of-months="2"
  class="rounded-lg border shadow-sm"
></hal-calendar>

With Date Constraints

<hal-calendar
  min-date="2025-01-10"
  max-date="2025-01-25"
  class="rounded-lg border shadow-sm"
></hal-calendar>

Without Outside Days

<hal-calendar
  show-outside-days="false"
  class="rounded-lg border shadow-sm"
></hal-calendar>

Week Starting on Monday

<hal-calendar
  week-starts-on="1"
  class="rounded-lg border shadow-sm"
></hal-calendar>

Attributes

Attribute Type Default Description
mode "single" | "multiple" | "range" "single" Selection mode
selected string (YYYY-MM-DD) - Selected date (single mode)
selected-dates string (comma-separated) - Selected dates (multiple mode)
range-start string (YYYY-MM-DD) - Range start date
range-end string (YYYY-MM-DD) - Range end date
default-month string (YYYY-MM-DD) today Initial month to display
min-date string (YYYY-MM-DD) - Minimum selectable date
max-date string (YYYY-MM-DD) - Maximum selectable date
number-of-months number 1 Number of months to display
week-starts-on number (0-6) 0 Day week starts on (0=Sunday)
show-outside-days boolean true Show days from adjacent months

Events

Event Detail Description
date-select { date: string } Fired when a date is selected (single mode)
dates-select { dates: string[] } Fired when dates change (multiple mode)
range-select { start: string | null, end: string | null } Fired when range changes
