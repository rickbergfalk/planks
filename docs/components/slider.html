Planks / Slider

Slider

An input where the user selects a value from within a given range.

Default

A basic slider with default settings. 

<plank-slider value="50"></plank-slider>

Various Values

0%
25%
50%
75%
100% 
<plank-slider value="0"></plank-slider>
<plank-slider value="25"></plank-slider>
<plank-slider value="50"></plank-slider>
<plank-slider value="75"></plank-slider>
<plank-slider value="100"></plank-slider>

Custom Range

Set custom min and max values.

<plank-slider value="50" min="0" max="200"></plank-slider>

Step Increment

Use the step attribute to change the increment value. 

<plank-slider value="50" step="10"></plank-slider>

Disabled

<plank-slider value="50" disabled></plank-slider>

API

Properties

Property Type Default Description
value number 0 Current value
min number 0 Minimum value
max number 100 Maximum value
step number 1 Step increment
disabled boolean false Whether the slider is disabled
orientation "horizontal" | "vertical" "horizontal" Slider orientation

Events

Event Detail Description
value-change { value: number } Fired when the slider value changes

Accessibility

  • Thumb has role="slider" for assistive technology
  • Supports aria-valuemin, aria-valuemax, and aria-valuenow
  • Arrow keys adjust value by step increment
  • Home/End keys jump to min/max values
  • Page Up/Down keys for larger increments
  • Disabled slider removes thumb from tab order
