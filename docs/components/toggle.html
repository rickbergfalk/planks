Toggle

A two-state button that can be either on or off.

Basic Usage

<hal-toggle aria-label="Toggle bold">
  <svg>...</svg>
</hal-toggle>

States

B Off
B On 
<hal-toggle aria-label="Toggle">B</hal-toggle>
<hal-toggle pressed aria-label="Toggle">B</hal-toggle>

Variants

B Default
B Outline 
<hal-toggle>B</hal-toggle>
<hal-toggle variant="outline">B</hal-toggle>

Sizes

B Small
B Default
B Large 
<hal-toggle size="sm">B</hal-toggle>
<hal-toggle>B</hal-toggle>
<hal-toggle size="lg">B</hal-toggle>

Disabled

B Disabled 
<hal-toggle disabled>B</hal-toggle>

With Text

Italic 
<hal-toggle aria-label="Toggle italic">
  <svg>...</svg>
  Italic
</hal-toggle>

API

Property Type Default Description
pressed boolean false The controlled pressed state of the toggle
variant "default" | "outline" "default" The visual variant of the toggle
size "sm" | "default" | "lg" "default" The size of the toggle
disabled boolean false Disables the toggle

Events

Event Detail Description
pressed-change boolean Fired when the toggle state changes
