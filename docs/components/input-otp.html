Input OTP

Accessible one-time password component with copy paste functionality.

API

hal-input-otp

Property Type Default Description
max-length number 6 Maximum number of characters
value string "" Current OTP value
pattern string - Regex pattern for validation (e.g., "^\d+$" for digits only)
input-mode "numeric" | "text" "numeric" Keyboard input mode
disabled boolean false Disables the input

Events

Event Detail Description
change { value: string } Fired when value changes
complete { value: string } Fired when all slots are filled

Methods

Method Description
focus() Focuses the input
clear() Clears the value

hal-input-otp-slot

Property Type Default Description
index number 0 Position of this slot in the OTP value
