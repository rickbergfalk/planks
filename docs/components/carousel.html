Carousel

A carousel with motion and swipe built using Embla Carousel.

Basic Usage

A simple horizontal carousel with navigation buttons.

1
2
3
4
5 
<plank-carousel class="w-full max-w-xs">
  <plank-carousel-content>
    <plank-carousel-item>
      <plank-card>...</plank-card>
    </plank-carousel-item>
    <plank-carousel-item>...</plank-carousel-item>
  </plank-carousel-content>
  <plank-carousel-previous></plank-carousel-previous>
  <plank-carousel-next></plank-carousel-next>
</plank-carousel>

Vertical Orientation

Set orientation to "vertical" to create a vertical carousel.

1
2
3 
<plank-carousel orientation="vertical" class="w-full max-w-xs">
  <plank-carousel-content class="h-[200px]">
    <plank-carousel-item>...</plank-carousel-item>
  </plank-carousel-content>
  ...
</plank-carousel>

Components

Component Description plank-carousel The carousel container with Embla integration plank-carousel-content Container for the carousel slides plank-carousel-item Individual carousel slide plank-carousel-previous Button to navigate to previous slide plank-carousel-next Button to navigate to next slide

plank-carousel

Property Type Default Description orientation "horizontal" | "vertical" "horizontal" Direction of the carousel scrolling

Methods

Method Description scrollPrev() Scroll to the previous slide scrollNext() Scroll to the next slide getApi() Get the underlying Embla Carousel API

Events

Event Description select Fired when the selected slide changes

Accessibility

  • Uses role="region" with aria-roledescription="carousel"
  • Each slide has role="group" with aria-roledescription="slide"
  • Navigation buttons have screen reader text
  • Supports keyboard navigation with ArrowLeft/ArrowRight
  • Previous/Next buttons are disabled when at the start/end of slides
