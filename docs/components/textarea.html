Textarea

Displays a multi-line text input field.

Default

<plank-textarea placeholder="Enter your message..."></plank-textarea>

Fixed Height

The textarea uses field-sizing: content to auto-expand with content. To set a fixed height, use Tailwind height classes. 

<plank-textarea placeholder="Fixed height textarea" class="h-40"></plank-textarea>

With Label

Bio 
<plank-label for="bio">Bio</plank-label>
<plank-textarea id="bio" placeholder="Tell us about yourself..."></plank-textarea>

Disabled

<plank-textarea placeholder="Disabled textarea" disabled></plank-textarea>

API

Property Type Default Description
placeholder string "" Placeholder text
value string "" Textarea value
disabled boolean false Disables the textarea
